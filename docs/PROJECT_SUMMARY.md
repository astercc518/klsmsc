## 🎯 项目概述

本项目是一个**企业级国际短信网关系统**，支持多通道、多国家计费、自定义发送方ID，并创新性地集成了**Telegram Bot**，实现零门槛注册和移动端发送能力。

### 核心创新点

1. ✅ **通道多国家计费**: 支持按国家、运营商、通道三级差异化计费
2. ✅ **可选SID发送**: 4级优先级智能匹配发送方ID
3. 🆕 **Telegram Bot集成**: 通过聊天机器人实现账号自动创建和短信发送

## 🎯 核心功能实现

### 1. 通道多国家计费 ⭐⭐⭐

**实现方案**:
```python
# 3级计费规则匹配
1. 通道 + 国家 + 运营商（MCC/MNC）
2. 通道 + 国家
3. 通道默认价格

# 长短信自动拆分计费
GSM-7: 160字符/条，超出按153字符/条
UCS-2: 70字符/条，超出按67字符/条

# 支持多币种
USD / CNY / EUR / GBP
```

**数据库设计**:
```sql
CREATE TABLE country_pricing (
  channel_id INT,
  country_code VARCHAR(10),
  mcc VARCHAR(10),
  mnc VARCHAR(10),
  operator_name VARCHAR(100),
  price_per_sms DECIMAL(10,4),
  currency VARCHAR(10),
  effective_date DATE,
  ...
);
```

**API示例**:
```python
GET /api/v1/pricing/calculate
{
  "phone_number": "+8613800138000",
  "message": "Hello World",
  "channel": "channel_cn_01"
}

Response:
{
  "price_per_sms": 0.05,
  "message_count": 1,
  "total_cost": 0.05,
  "currency": "CNY"
}
```

### 2. 可选SID发送 ⭐⭐⭐

**实现方案**:
```python
# 4级优先级匹配
1. 用户API指定的SID（需验证权限）
2. 账户+国家+通道专属SID
3. 账户+通道通用SID
4. 通道默认SID

# 各国SID规则适配
• 美国: 字母SID，最长11位
• 中国: 数字SID，如106开头
• 印度: 6位DLT模板ID
• 英国: 字母SID，需预注册
```

**数据库设计**:
```sql
CREATE TABLE sender_ids (
  sid VARCHAR(20),
  account_id INT,
  channel_id INT,
  country_code VARCHAR(10),
  status ENUM('pending','active','rejected'),
  is_default BOOLEAN,
  ...
);
```

**API示例**:
```python
POST /api/v1/sms/send
{
  "phone_number": "+8613800138000",
  "message": "Your code is 123456",
  "sender_id": "MyApp"  // 可选，不填自动匹配
}
```

### 3. Telegram Bot集成 ⭐⭐⭐ 🆕

**核心价值**:
- **零门槛注册**: 无需访问网站，Telegram内完成
- **便捷发送**: 聊天窗口发送短信
- **实时通知**: 送达状态自动推送
- **移动优先**: 随时随地使用

**技术实现**:
```python
# Bot框架
python-telegram-bot 20.7

# 核心命令
/start     - 启动Bot
/register  - 注册账号（自动创建）
/send      - 发送单条短信
/batch     - 批量发送（上传CSV）
/balance   - 查询余额
/records   - 发送记录
```

**注册流程**:
```
1. 用户发送 /register
2. Bot请求输入邮箱
3. 调用后端API创建账号
4. 自动生成API密钥
5. 赠送10条免费短信
6. 返回账号信息
```

**发送流程**:
```
1. 用户发送 /send
2. Bot请求输入手机号（验证格式）
3. Bot请求输入短信内容
4. 显示预览和费用
5. 用户确认
6. 调用后端API发送
7. 返回消息ID
8. 送达后自动推送通知
```

**数据库扩展**:
```sql
CREATE TABLE telegram_users (
  telegram_id BIGINT UNIQUE,
  telegram_username VARCHAR(100),
  account_id INT,
  notification_enabled BOOLEAN,
  ...
);
```

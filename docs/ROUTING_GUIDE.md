# 通道路由规则配置指南

## 概述

路由规则用于指定「国家 → 通道」的映射。当客户发送短信时，系统根据目标号码的国家/区号选择通道。

## 配置入口

**管理后台 → 通道管理 → 点击通道的「路由配置」按钮**

## KAOLA_TH 等多国家通道配置

若通道（如 KAOLA_TH）支持多个国家，需为每个国家添加一条路由规则：

1. 进入 KAOLA_TH 通道的「路由配置」
2. 点击「添加路由规则」
3. 选择国家（如：泰国、菲律宾、越南、印度尼西亚等）
4. 设置优先级（数字越大优先级越高，同国家多通道时使用）
5. 保存

### 自动配置路由

**配置价格时自动创建路由**：在通道的「费率配置」中新增某国家的价格时，若该通道+国家尚无路由规则，系统会自动创建一条路由（优先级 0、启用）。无需手动再添加路由。

### 示例：KAOLA_TH 支持泰国、菲律宾、越南

| 国家   | 国家代码 | 优先级 |
|--------|----------|--------|
| 泰国   | TH       | 100    |
| 菲律宾 | PH       | 100    |
| 越南   | VN       | 100    |

每条规则单独添加，共 3 条。或通过**费率配置**添加对应国家价格，路由会自动创建。

## 国家代码格式

系统**同时支持**两种格式，任选其一即可：

- **ISO 国家码**：TH（泰国）、PH（菲律宾）、VN（越南）、ID（印度尼西亚）等
- **电话国码**：66（泰国）、63（菲律宾）、84（越南）、62（印度尼西亚）等

推荐使用 ISO 码（TH、PH、VN），更直观。

## 优先级说明

- 同一国家若配置了多个通道，**优先级高的通道**优先使用
- 账户若已绑定通道，则仅在绑定通道中按优先级选择
- 建议同一通道的多国家规则使用相同优先级（如 100）

## 注意事项

1. 添加路由规则后，需确保该通道在**通道-国家关系**中已配置对应国家（若有此功能）
2. 路由结果会缓存约 5 分钟，新增/修改规则后会自动失效缓存
3. 账户绑定通道后，只会使用绑定通道进行路由，未绑定的通道不会参与
